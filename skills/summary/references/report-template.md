# 한화이글스 요약 레포트 템플릿

> 이 파일은 최종 레포트 생성 시 참조하는 구조 템플릿이다.
> 실제 데이터로 치환하여 사용한다.

---

## 레포트 헤더

```markdown
# 2026 한화이글스 뉴스 요약 레포트

> **보고 일자:** {YYYY-MM-DD}
> **수집 범위:** 최근 24시간
> **독자:** 야구 감독님
> **작성:** summary팀 (클로니·애나·수미·퀴니)
> **기준:** 2026 신한 SOL KBO리그 시즌 진행 중 (Fact/Opinion 분리)
```

---

## 시즌 현황 스냅샷 (선택 — 데이터 확인 가능 시 포함)

```markdown
## 📊 시즌 현황 ({기준일} 현재)

| 항목 | 수치 |
|------|------|
| 성적 | {N}승 {N}무 {N}패 |
| 순위 | **{N}위** |
| 승률 | {0.NNN} (승차 {N}) |
| 팀 타율 | {0.NNN} |
| 팀 평균자책 | {N.NN} |

> **핵심:** {한 줄 요약}
```

---

## 뉴스 섹션 (5건 필수)

### 해외 뉴스 (2건)

```markdown
## 📰 [해외] 뉴스 1 — {제목}

**출처:** {언론사} ({날짜})
**URL:** {URL}
**원문 언어:** {영어/일본어 등} → 한국어 번역

{번역 요약 — 500자 이내, 명사체}

> **감독님께:** {전술·라인업 관점 인사이트 1줄}
```

### 국내 뉴스 (3건)

```markdown
## 📰 [국내] 뉴스 {N} — {제목}

**출처:** {언론사} ({날짜})
**URL:** {URL}

{요약 — 500자 이내, 명사체}

> **감독님께:** {전술·라인업 관점 인사이트 1줄}
```

---

## 팬 여론 종합 (Opinion)

```markdown
## 💬 팬 여론 종합

| 구분 | 내용 |
|------|------|
| 긍정 | {긍정 반응 요약} |
| 부정 | {부정 반응 요약} |
| 불안 | {불안 요소 요약} |
| 요구 | {팬들의 주요 요구사항} |
```

---

## 핵심 인사이트 (감독님께)

```markdown
## ⚾ 감독님께 드리는 핵심 인사이트

1. **{항목}** — {내용}
2. **{항목}** — {내용}
3. **{항목}** — {내용}
```

---

## 출처 목록 (푸터)

```markdown
---

*출처 목록*
- [{언론사} - {기사 제목}]({URL})
- [{언론사} - {기사 제목}]({URL})
...
```

---

## 작성 규칙 체크리스트

- [ ] 뉴스 5건 (해외 2 + 국내 3) 포함
- [ ] 해외 뉴스 한국어 번역 완료
- [ ] 각 뉴스 500자 이내
- [ ] Fact는 출처 URL 포함
- [ ] Opinion은 "(팬 반응)" 명시
- [ ] 수치 미확인 항목은 "(※ 원문 미확인)" 표기
- [ ] 감독님 관점 인사이트 포함
- [ ] 명사체 사용 (예: "~함", "~임", "~됨")
