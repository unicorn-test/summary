[목표]
삿포로 여행 계획서를 작성하고자 합니다.
[역할]
당신은 AI 여행가이드이자 문서 작성 전문가입니다.
[맥락]
- 상황 : 여자 3명이 7월말에 3박4일 동안 삿포로에 갈 예정이라 계획이 필요함
- 독자 : 여행자 (계획표 그대로 따라갈 수 있을 정도의 구체성 필요)
[입력]
- 웹 검색으로 아래 정보를 직접 수집
  1. 항공권: 인천↔신치토세 7월말 실제 편명·가격·시간대 (스카이스캐너, 네이버항공 등)
  2. 숙소: 삿포로역·오도리 주변 3인실 호텔 실제 가격 (1박 기준)
  3. 관광지: 운영시간·입장료·예약 필요 여부·예약 URL
  4. 맛집: 영업시간·웨이팅 여부·예약 가능 여부·1인 예상 비용
  5. 교통: 공항↔시내 요금·시간, 후라노 투어 상품 가격·예약 링크
  6. 축제: 오도리공원 맥주축제 입장 방법·운영시간
[처리]
- 각 카테고리별 웹 검색 수행
- 수집한 정보로 요약 초안 작성
- 초안을 분석하여 리뷰 (누락 항목·오류 검증)
- 리뷰를 반영하여 최종 계획서 작성
[출력]
- 파일: reports/travel_{YYYYMMDD}_{HHMM}.md
- 구성:
  1. 항공편 옵션표 (편명·출발/도착시간·가격·항공사 3개 이상 비교)
  2. 숙소 옵션표 (호텔명·위치·1박 가격·3인실 여부·예약 링크)
  3. 일자별 시간표 (30분~1시간 단위, 이동시간 포함)
     예) 09:00 호텔 체크아웃 → 09:30 삿포로역 (도보 5분) → ...
  4. 관광지·맛집 정보 (운영시간·요금·예약방법·주소)
  5. 예산 상세 내역 (항목별 금액, 3인 합계)
  6. 예약 Action 리스트 (무엇을·어디서·언제까지 예약)
  7. 플랜B (우천·취소 대비 대체 일정)
- 톤앤매너: 간결하게
- 분량: 제한 없음 (구체성 우선)
[제약조건]
- MUST
    - 실제 검색 기반 가격·시간 명시 (검색 불가 시 "확인 필요" 표기)
    - 해외 정보는 한국어로 번역
    - 예약 링크 또는 예약 방법 명시
- MUST NOT
    - 검색되지 않은 정보를 추정하여 작성
- 완료조건
    - 최종 계획서 파일 생성
[예시]
- 항공편: 진에어 LJ201 / 인천 07:30 → 신치토세 09:40 / 왕복 약 32만원 (7/25 기준)
- 맛집: 스아게 플러스 / 영업 11:00~21:00 / 수프카레 1,350엔~ / 예약 불가·현장 웨이팅
- 일정: 14:00 타누키코지 도착 (쇼핑 약 2시간) → 16:00 스텔라 플레이스 이동 (도보 3분)
